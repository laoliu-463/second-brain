# 第二大脑快速启动指南

> 本指南帮助你快速完成最后配置并开始学习

---

## 第一步：Obsidian 内配置（15分钟）

### 1.1 安装插件

打开 Obsidian → 设置 → 社区插件 → 关闭安全模式 → 安装以下插件：

| 插件 | 功能 |
|------|------|
| Templater | 模板自动插入 |
| Linter | 格式自动修正 |
| Obsidian Git | Git自动同步 |
| Dataview | 双向链接自动化 |

### 1.2 配置 Templater

1. 社区插件 → Templater → 启用
2. 模板文件夹填写：`09-模板`
3. 勾选「在任意文件夹启用」（可选）

### 1.3 配置 Linter

1. 社区插件 → Linter → 启用
2. 开启所有格式化规则（见插件配置指南）
3. 设置快捷键：`Ctrl+Alt+L`

### 1.4 配置 Git

1. 社区插件 → Obsidian Git → 启用
2. 设置 → 自动提交：30分钟
3. 设置 → 自动推送：启用
4. 设置 → 启动时拉取：启用
