# 测试样例集说明

本目录用于验证以下规则文件是否稳定可执行：

- `00-系统/AI整理规则.md`
- `00-系统/整理检查清单.md`
- `00-系统/审核模式输出格式.md`

## 当前覆盖范围

- 样例总数：12
- 常规场景：项目、日常、阅读、资源、灵感、长期知识候选
- 边界场景：多标签冲突、来源不足、链接冲突、暂不确定、内容混杂、分类与语义冲突

## 使用方式（建议）

1. 从 `样例01` 到 `样例12` 逐篇执行 AI 整理（默认 `review` 模式）
2. 每篇整理后使用《整理检查清单》打分
3. 使用《审核模式输出格式》核对输出结构
4. 记录失败点并迭代规则

## 最低通过标准

- 不伪造参考链接
- 不误链
- 不越权执行高风险动作
- 分类与目录决策可解释
- 来源不足时能降级处理
